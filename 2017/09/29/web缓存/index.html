web缓存机制


web缓存类型：


  • 数据库缓存

  • 服务器缓存

    • 代理服务器缓存、CDN缓存
  • 浏览器缓存

浏览器缓存

http缓存：
通过网络获得较大的资源速度缓慢而且开销很大，所以重复利用之前的资源成为性能优化的一个重要的方面。

制定缓存策略:
1.是否需要缓存：不需要:no-store
2.是否允许直接从缓存中读取：不允许：no-cache
3.是否可以被任何人缓存 是：public 否：private
4.通过max-age、Etags/if-no-match、last-modifed/if-modifed-since来缓存

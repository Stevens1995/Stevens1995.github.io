函数的双重用法

javascript函数有两个内部方法，[[call]]和[[constructor]]，当函数不用new关键字调用时，[[call]]方法运行，执行函数体中的方法。当函数使用new关键字时，[[constructor]]方法调用，创造一个新对象，成为new target，然后将函数的this指向新对象，运行函数体。

并不是所有的函数都有[[constructor]]方法，比如箭头函数就没有
